## 🩺 Overview

**MediMart** is a modern full-stack **medicine e-commerce platform** that provides a fast, secure, and user-friendly online pharmacy experience.
Users can browse medicines, place orders, and manage profiles, while admins can efficiently control inventory, customers, and orders.

---

## 🚀 Features

### 👤 Customer

* 🔍 Search medicines with filters
* 🛒 Add to cart & checkout
* 📦 Order tracking
* 🎟️ Apply coupon codes
* 📍 Multiple saved addresses
* 💳 Payment options (COD & UPI)

### 🧑‍💼 Admin

* 📊 Dashboard overview
* 📦 Manage orders (view/delete)
* 👥 Manage customers
* 💊 Add / Update / Delete medicines
* 🎟️ Manage coupons & offers

### 🌗 UI/UX

* 🌙 Light & Dark mode
* ⚡ Fast & responsive design
* 📱 Mobile-friendly interface

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/Loop213/MediMart.git
cd MediMart
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Environment Variables (.env)

```
PORT=3000
MONGO_URI=your_mongodb_connection
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_KEY=your_key
CLOUDINARY_SECRET=your_secret
```

### 4. Run Project

```bash
npm start
```
